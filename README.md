# Talent Acquisition Specialist AI Persona

## Overview

This repository contains a JSON-based persona for an AI agent designed to emulate a **Senior Talent Acquisition Specialist**. 

The persona, named **Alex Rivera**, is built for use in:

- Recruitment chatbots and virtual assistants
- HR tech demos and interview simulation tools
- AI copilots for hiring teams
- Training and onboarding for junior recruiters

---

## Persona Highlights

| Attribute | Description |
|-----------|-------------|
| **Name** | Alex Rivera |
| **Role** | Senior Talent Acquisition Specialist |
| **Company** | Global Tech Corp (SaaS, 500+ employees) |
| **Core Focus** | Full-cycle recruitment, employer branding, data-driven hiring |
| **Communication** | Professional, empathetic, and action-oriented |

---

## Persona Structure (JSON)

The JSON is organized into the following key sections:

- **Core Traits** – The fundamental characteristics that define Alex’s approach.
- **Communication Style** – Tone, formality, and empathy markers.
- **Knowledge Areas** – Domains of expertise (legal, operational, strategic).
- **Hard & Soft Skills** – Technical and interpersonal capabilities.
- **Decision-Making Framework** – Priorities, risk tolerance, and data reliance.
- **Typical Scenarios** – Common challenges and how Alex responds.
- **Response Guidelines** – Rules for candidate interactions and manager advising.
